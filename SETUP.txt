─── ONE MANDATORY SUPABASE DASHBOARD STEP ──────────────────────
Go to: https://supabase.com/dashboard/project/lzvopmmasfvwtepcrpkw/auth/templates

Click "Confirm signup" and change the email body from:
  <a href="{{ .ConfirmationURL }}">Confirm your email</a>
To:
  <p>Your verification code: <strong>{{ .Token }}</strong></p>
  <p>It expires in 2 minutes. Do not share it.</p>

This makes Supabase send a 6-digit OTP code instead of a
magic link, which is what register.html expects.

─── CREATE YOUR FIRST ADMIN ────────────────────────────────────
1. Open register.html in browser
2. Choose "Student" (any role — you'll change it manually)

   Find your row → change role from 'client' to 'Admin'

─── HOW AUTH WORKS (no Flask) ──────────────────────────────────
SIGN UP:
  register.html → supabase.auth.signUp() → Supabase sends OTP
  User enters OTP → supabase.auth.verifyOtp()
  DB trigger (handle_new_user) → creates public.users row
  User redirected to login.html

LOGIN:
  login.html → supabase.auth.signInWithPassword()
  → reads role from public.users → redirects to correct dashboard

PASSWORD RESET:
  forgot-password.html → supabase.auth.resetPasswordForEmail()
  → OTP sent → verified → supabase.auth.updateUser(password)
